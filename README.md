# Music仿音乐项目
## 包含管理者页面（仿虾米音乐界面风格）及音乐项目主页（自主编写界面风格）
### 本项目涉及以下内容及框架
---------------------------------------------------------------------------------------------------------
* 音乐存储：七牛（本人用node自建立一个服务器用以返回uptoken的AccessKey及AccessId进行管理者界面上传七牛存储音乐）
* 数据存储：LeanCloud
* jQuery/原生JS
* 轮播图：Swiper
* 项目使用rem自适应布局、登录页及主页页面间数据使用cookie传递
* 使用MVC模型编写JS代码（包括使用发布订阅模式用以模型间交互）
